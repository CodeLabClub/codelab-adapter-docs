# readme.md
[CodeLab Adapter](https://adapterv2.codelab.club) 的项目文档，基于 [mkdocs](https://www.mkdocs.org/)。

欢迎大家一起来维护文档。


# Usage

```
git clone https://github.com/CodeLabClub/codelab-adapter-docs
cd codelab-adapter-docs
pip3 install mkdocs
# debug
mkdocs serve
# build
mkdocs build

# publish
make push
```
