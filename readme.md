# readme.md
[codelab-adapter](https://codelab-adapter-docs.codelab.club/)的项目文档，基于[mkdocs](https://www.mkdocs.org/)。

欢迎大家一起来维护文档。


# Usage

```
pip3 install mkdocs
# debug
mkdocs serve
# build
mkdocs build

# publish
make push
```
