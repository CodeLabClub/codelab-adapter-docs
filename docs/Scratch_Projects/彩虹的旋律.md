# 彩虹 🌈 的旋律

<video width=80% src="/video/scratch_rainrow.mp4" controls="controls"></video>

## 项目介绍

将彩虹糖用作音符来创作音乐。

## 思路

- 彩虹糖在桌子上排成一列，使用摄像头对着它们, 彩虹糖被投影到 Scratch 舞台；
- 一个粉色箭头在屏幕中间，从左向右滑动；
- 当粉色箭头碰到不同颜色的彩虹糖时，演奏不同旋律的音符。
