# Scratch PPT

使用 Scratch 来制作 PPT，用于分享和演讲。

## 思路

将舞台背景用作 PPT 载体，绘制若干舞台背景，演讲时进行切换。

!!!tip
    使用 **角色** 让演讲内容生动起来。

## 涉及的知识

- **外观** 积木
    - 背景切换
- 制作舞台背景
- 事件
- Scratch EIM 插件
- CodeLab Adapter Python Kernel Extension
    - 打开本地文件(目录/视频)

## 进阶

- 使用 micro:bit 制作翻页笔
- 制作眼动仪翻页笔，帮助双手不便利的人

## 想象空间

- 使用 Switch Joy-Con 作为翻页笔
