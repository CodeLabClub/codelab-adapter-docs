# Overdrive

## 介绍
![](/img/overdrive.jpg)

>  与你的朋友或人工智能来一场速度与激情的较量。

Overdrive 是一款可编程赛车，内置多种传感器，在特制的跑道上运行时能定位到自身的位置。此扩展可以实时控制通过蓝牙连接的赛车。

## hello world
打开 Scratch overdrive 插件。

使用连接图标，连接小车。

跑起来:
