Tutorial

安装依赖

我的环境是: MacOS blender2.8

cd /Applications/blender.app/Contents/Resources/2.80/python/bin
wget https://bootstrap.pypa.io/get-pip.py
./python3.7m ./get-pip.py --prefix /Applications/blender.app/Contents/Resources/2.80/python
./python3.7m pip3 install pyzmq

在blender中运行blender_server

在blender中运行blender_server.py

在blender2.79b中， 打开Text Editor，运行blender_server.py

在blender2.8中,我喜欢打开Scripting标签页(使用Text Editor也可以), 运行blender_server.py:

在codelab-adapter运行extension_blender.py插件

下载https://github.com/Scratch3Lab/codelab_adapter_extensions/blob/master/extension_blender.py插件。

下载完成后，重启codelab-adapter.

开始使用

打开scratch3.codelab.club， 开始使用。

如果你在网页里没看到blender插件，请刷新浏览器缓存。
